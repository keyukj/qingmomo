## 🚀 快速测试流程

### 步骤1: 打开应用

```
1. 点击应用图标
2. 等待应用启动
3. 登录用户账号
```

### 步骤2: 进入充值页面

```
1. 进入个人资料页面
2. 找到"充值"或"钱包"选项
3. 点击进入充值页面
```

### 步骤3: 选择套餐

```
1. 看到充值套餐列表
2. 选择任意套餐（建议选最便宜的）
3. 点击套餐
```

### 步骤4: 验证加载状态

```
✅ 应该看到:
   - 套餐上显示转圈圈加载指示器
   - 套餐变为灰色（不可点击）
   - 价格位置显示加载动画
```

### 步骤5: 登录沙盒账号

```
1. 弹出App Store登录界面
2. 输入沙盒测试账号
3. 输入密码
4. 点击登录
```

### 步骤6: 验证继续加载

```
✅ 应该看到:
   - 转圈圈继续显示
   - 没有任何错误提示
   - 应用没有卡住
```

### 步骤7: 完成支付

```
1. 根据提示完成支付
2. 可能需要输入密码或使用Face ID
3. 等待支付完成
```

### 步骤8: 验证成功

```
✅ 应该看到:
   - 绿色成功提示: "成功充值 X 金币"
   - 金币余额增加
   - 转圈圈消失
   - 套餐恢复可点击状态
```

## ❌ 错误处理测试

### 测试1: 取消登录

```
1. 点击套餐
2. 看到加载指示器
3. 弹出登录界面
4. 点击"取消"
```

```
2. 看到加载指示器
3. 弹出登录界面
4. 输入错误的账号

```

```
1. 点击套餐

3. 弹出登录界面
```

## 🔍 调试技巧

### 查看日志

```bash
# 在Xcode中查看控制台输出
# 打开Xcode → Window → Devices and Simulators
# 选择设备 → 查看Console
```

### 常见问题

**问题**: 加载指示器不显示
```
解决: 
1. 检查_isPurchasing状态是否更新
2. 检查onPurchaseStarted回调是否被调用
3. 查看Xcode日志
```

**问题**: 金币没有增加
```
解决:
1. 检查CoinService.addCoins()是否被调用
2. 检查SharedPreferences是否正常
3. 检查产品ID对应的金币数
```

**问题**: 一直显示加载
```
解决:
1. 等待30秒看是否显示超时
2. 检查网络连接
3. 尝试重新登录
```
