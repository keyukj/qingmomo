## 🔍 如果出现问题

### 问题1: 加载指示器不显示

**检查清单**:
1. 确认 `_isPurchasing` 状态是否更新
2. 确认 `onPurchaseStarted` 回调是否被调用
3. 查看Xcode控制台日志

**解决方案**:
```dart
// 在 _setupPurchaseListener 中添加日志
IAPService.onPurchaseStarted((productId) {
  print('购买开始: $productId');
  setState(() {
    _isPurchasing = true;
  });
});
```

### 问题2: 金币没有增加

**检查清单**:
1. 确认 `CoinService.addCoins()` 是否被调用
2. 确认 `SharedPreferences` 是否正常工作
3. 确认产品ID对应的金币数是否正确

**解决方案**:
```dart
// 在 _handleSuccessfulPurchase 中添加日志
print('添加金币: $coins');
await CoinService.addCoins(coins, reason: '充值 - $productId');
```

### 问题3: 一直显示加载

**检查清单**:
1. 等待30秒看是否显示超时
2. 检查网络连接
3. 尝试重新登录

**解决方案**:
- 检查网络连接是否正常
- 确认沙盒测试账号是否正确
- 查看Xcode日志中的错误信息
